# exercism-tracks

These are my [exercism.io](https://exercism.io) tracks

## Installation

Run

```bash
brew bundle
```

To install requirements

## Running on docker

Load utility functions and bring docker up
```bash
source ./exercism-tracks.sh
exercism_start
```
